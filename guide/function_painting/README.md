# Function Painting

> 漂亮的函数生成器

By NriotHrreion

## 介绍

这是一个基于**javascript**与**h5 canvas**运行的函数图像生成器

如果你要使用它, 你可以进入这个网址

[https://nin.red/using/function-painting/launch.html](/using/function-painting/launch.html)

当然, 如果你想要在浏览器书签栏中添加某个函数图像, 你可以在浏览器中添加:

`https://nin.red/using/function-painting/?fx={你要添加的函数}`
