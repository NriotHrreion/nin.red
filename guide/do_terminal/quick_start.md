# 快速开始

By NriotHrreion

## 环境

因为**Do-Terminal**是使用node编写的

所以运行它之前, 你的电脑必须得安装[Node.js](https://nodejs.org)

## 使用

下载Do Terminal到你的电脑上

```bash
$ git clone https://github.com/NriotHrreion/Do-Terminal.git
```

接着进入工程根目录, 下载依赖

```bash
$ npm install
```

在`localhost`上部署服务器

```bash
$ node index
```

输出以下内容即运行成功

```
[00:00:00] $ Server Started
```
