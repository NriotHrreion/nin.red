# Function Painting

> A beautiful function painter

By NriotHrreion

## Description

This is a function painter which running with **javascript** and **h5 canvas**

If you want to use it, please go to

[https://nin.red/using/function-painting/launch.html](/using/function-painting/launch.html)

Of course, if you want to add a function to your browser's bookmarks, you can add:

`https://nin.red/using/function-painting/?fx={The function}`
