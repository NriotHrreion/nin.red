# Quick Start

By NriotHrreion

## Enviroment

Because **Do-Terminal** is written in Node

So you must install [Node.js](https://nodejs.org), before you run the terminal.

## Usage

Download the Do Terminal.

```bash
$ git clone https://github.com/NriotHrreion/Do-Terminal.git
```

Then `cd` to the project folder, and do

```bash
$ npm install
```

Run it on `localhost`

```bash
$ node index
```

If it output these info, Complete!

```
[00:00:00] $ Server Started
```
