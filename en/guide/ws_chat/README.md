# Online Chatting Room

> A simple `Nodejs` + `Websocket` Chatting Room

By NriotHrreion

## Description

The Online Chatting Room works with `Node.js` and `WebSocket`, and it supports sending messages or photos.

You can serve the chatting room on a server, and chat with your friends in the chatting room.

## Usage

Before using it, you need to install [Node.js](https://nodejs.org).

First, download the online chatting room to your computer.

```bash
$ git clone https://github.com/NriotHrreion/chat-server.git
```

Then, `cd` to the project folder, install packages.

```bash
$ npm install
```

Run it on `localhost`.

```bash
$ node index
```

Type the address in browser, and join the chatting room.

```
http://localhost:8920
```

!> **Attention!** The Online Chatting Room is using two ports (**8920/3500**), if other applications are using, please change them.
